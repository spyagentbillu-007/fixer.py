# Script Checker Tool

**Script Checker** ek lightweight aur easy-to-use tool hai jo aapki Python scripts ko check karta hai aur errors/warnings detect karta hai. Ye Termux aur desktop Python environments dono me kaam karta hai.

## Features

- ✅ Script ka syntax check  
- ✅ Errors aur warnings highlight kare  
- ✅ Easy-to-use command line interface  
- ✅ Termux friendly
